# 🧠 Ivan — ML Research & Engineering Portfolio

> **Live portfolio:** [https://YOUR_USERNAME.github.io/ml-portfolio/](https://YOUR_USERNAME.github.io/ml-portfolio/)

A fully interactive, self-hosted portfolio tracking my 18-month journey to ML Research & Engineer — modelled on MIT 6.S191, Stanford CS229/CS224N, fast.ai, and deeplearning.ai.

## Features

- 📍 **Roadmap** — 5 phases, 72 weeks, every module with recruiter-facing deliverables
- ✅ **Task Tracker** — 54 tasks with checkbox completion, progress sliders, phase filters
- 🛠 **Projects** — 13 portfolio projects with repo structure, stack, and GitHub integration
- 📊 **Analytics** — Progress chart, skill heatmap, GitHub commit activity grid
- 📚 **Resources** — Curated books, courses, YouTube, tools, papers
- 🌙 **Dark/Light mode**
- ⚡ **GitHub API integration** — live commit tracking
- 📱 **Fully responsive** — laptop, iPad, iPhone

## Deploy your own

```bash
# 1. Fork or clone this repo
git clone https://github.com/YOUR_USERNAME/ml-portfolio.git

# 2. Edit js/data.js to personalise (name, start date, etc.)

# 3. Push to GitHub
git add . && git commit -m "🚀 Launch portfolio" && git push

# 4. Enable GitHub Pages: Settings → Pages → main branch → Save
# Your site is live at https://YOUR_USERNAME.github.io/ml-portfolio/
```

## Tech stack

Pure HTML/CSS/JavaScript — zero dependencies, zero build step. Deploys instantly to GitHub Pages.
